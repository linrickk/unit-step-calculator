Unit Step Function Calculator

Evaluator: u(t − a)

Enter values and press Compute.

Piecewise Builder: f(t) = c + Σ k·u(t − a)

Each term is k · u(t − a). Example: k=3, a=2 means +3 turns on at t=2.
Add terms, then click Build piecewise.

Graph: f(t) = c + A·u(t − a)

This graph is generated by Python (matplotlib) on the server and sent as an image.
